USB Library of Shared Sources

This module provides a set of useful header files for implementing USB devices.

Contents

These include the many of the defines from the USB standards, including:

  • Class codes
  • PID types
  • Audio class descriptors
  • Terminal types

Types

This module defines the following useful shared types:

  • USB_BmRequestType_t

    Defines the Recepient, Type and Direction of a USB request.

  • USB_SetupPacket_t

    Typedef for setup packet structure.
